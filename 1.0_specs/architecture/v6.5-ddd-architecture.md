### Dependency Rules

| Context | Depends On | Provides |
|---------|-----------|----------|
| WORLD | вҖ” (pure functions) | terrain, cities, countries |
| STATE | WORLD | G (global state) |
| ECONOMY | STATE | trade, stocks, companies |
| POLITICS | STATE | countryAI, diplomacy |
| COMBAT | STATE | battles, enemies |
| PLAYER | STATE | movement, skills |
| RENDER | STATE (read-only) | canvas drawing |

### Wire Map
```
ORC.init(role, config)
  вҶ’ WORLD.genTerrain(seed, mapType)

  вҶ’ STATE.init(player, world)
  вҶ’ ECONOMY.init(companies, aiCEOs)
  вҶ’ POLITICS.init(countries, factions)
  вҶ’ PLAYER.init(role)
  вҶ’ RENDER.init()

  вҶ’ PLAYER.update()     // movePlayer, mount decay
  вҶ’ ECONOMY.update()    // processFacilities, stocks, loans, tax
  вҶ’ POLITICS.update()   // countryAI, factions, revolutions
  вҶ’ COMBAT.check()      // random encounters, bounty hunters
  вҶ’ UI.update()         // HUD refresh
  вҶ’ RENDER.draw()       // canvas render via LAYERS

ORC.save() вҶ’ STATE.serialize()

```

### Anti-patterns (proibidos)
- вқҢ Funcao de ECONOMY chamando RENDER diretamente
- вқҢ Funcao de RENDER escrevendo em STATE
- вқҢ Acesso direto a G sem passar pelo contexto
- вқҢ Novo dominio sem spec correspondente
