# SPEC: Economy & Trade System v5.7

## Status: ✅ IMPLEMENTED

## Overview
Dynamic market economy with 110+ goods, supply/demand pricing, stock market, loans, and company ownership.

## Trade System

### Market (mdMarket)
- 110+ goods in 15 categories (food, material, metal, weapon, armor, magic, alchemy, tool, luxury, book, exotic, transport)
- Dynamic pricing: `price = basePrice × (1 + demand×0.003 - supply×0.003 + jitter)`
- Search filter + category tabs
- Buy/sell with quantity buttons (+1, +5, All)

### Functions
| Function | Description |
|----------|-------------|
| `buyGood(gid,qty)` | Purchase from city, cost = price × qty |
| `sellGood(gid,qty)` | Sell to city, revenue = price × qty × (1 + trading × 0.03) |
| `totWt()` | Total weight of inventory |

### Caravan System
- `buyCaravan()`: $500, +50 cargo capacity
- `upgradeCaravan()`: +25 cap for $300
- `hireGuard()`: +1 guard for $200
- Food spoilage in caravan (perishable goods rot)

## Stock Market

### Public Companies (G.comps)
- 8 auto-generated companies at world gen
- Stock prices fluctuate ±5% every 2 days
- 30-day price history chart

### Player Operations
| Function | Description |
|----------|-------------|
| `buyStock(cid,shares)` | Buy at current price |
| `sellStock(cid,shares)` | Sell at current price |
| `updateStocks()` | Update player-owned company values |

### CEO System
| Function | Description |
|----------|-------------|
| `foundCompany(sector)` | Create company ($20K, 10 sectors) |
| `expandCompany(idx,cityId)` | Expand to city ($5K + 1% pop) |
| `ipoCompany(idx)` | Go public ($50K) |
| `hostileTakeover(cid)` | 51% shares = company becomes yours |

## Loans
| Function | Description |
|----------|-------------|
| `takeLoan(amount)` | Borrow at 15% interest, 30-day term |
| `processLoans()` | Daily payment deduction |
| `payLoanFull(lid)` | Pay remaining balance |

## Goods Catalog (GD)
- 15 categories, 110+ items
- Fields: id, n (name), p (price), w (weight), cat (category)
- Optional: per (perish days), atk, def, dur, eq (equip slot), use (consumable), r (rarity)
- Indexed lookup: `GI[gid]` for O(1) access
