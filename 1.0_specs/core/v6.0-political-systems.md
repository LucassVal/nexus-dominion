# SPEC: Political Systems & Corporate Sovereignty v6.0

## Status: 📝 SPEC (awaiting approval)

## Overview
Major expansion adding political depth to kingdoms, corporate entities owned by AI and kingdoms, territory borders with operating licenses, and diverse government types with gameplay effects.

---

## 1. GOVERNMENT TYPES (replaces simple gov string)

Each kingdom now has a rich government structure:

| Type | ID | Legislative | Tax Rate | Stability | Corporate |
|------|-----|-------------|----------|-----------|-----------|
| **Monarchy** | monarchy | Crown decree | 20-35% | High | Royal monopolies |
| **Parliamentary** | parliament | Elected body | 10-20% | Medium | Open market |
| **Theocracy** | theocracy | Religious council | 15-25% | High | Tithe-based |
| **Technocracy** | technocracy | Expert committee | 5-15% | Very High | Innovation grants |
| **Feudal Barony** | barony | Baron + vassals | 25-40% | Low | Land-grant only |
| **Merchant Republic** | merchant_rep | Trade guilds | 8-15% | Medium | Free enterprise |
| **Military Junta** | junta | High command | 15-30% | High | Arms contracts |
| **City-State** | city_state | Citizen assembly | 5-12% | Medium | Open to all |

### Government Properties
```javascript
country.gov = {
  type: 'monarchy',        // government type
  name: 'Kingdom of Aurelia',
  taxRate: 25,             // base tax %
  stability: 75,           // 0-100
  corpPolicy: 'monopolies', // corporate policy
  tradePolicy: 'regulated', // free/regulated/restricted
  migrationPolicy: 'open',  // open/controlled/closed
  termLength: 0,            // 0 = lifetime (monarchy), N = years
  nextElection: 0,          // day of next election
  factions: []              // internal political factions
}
```

## 2. TERRITORY & BORDERS

### Territory System
- Each kingdom controls specific tiles on the map
- Territory = cities + surrounding tiles (radius based on city size)
- Border disputes when territories overlap
- Companies need **operating licenses** per territory

### Operating License
```javascript
license = {
  id: 'lic_001',
  company: 'Apex Dynamics',
  territory: 'Aurelia',
  type: 'full',         // full/restricted/trade_only
  cost: 5000,           // license fee
  taxRate: 12,          // territory tax on company revenue
  expiry: 120,          // game days
  granted: 45           // day granted
}
```

### Border Mechanics
- Borders visible on map (dashed colored lines)
- Cross-border trade: tariffs apply
- Companies without license: can't operate in territory
- Smuggling: possible but risks penalties

## 3. CORPORATE ENTITIES (expanded)

### Ownership Types
| Type | Owner | Examples |
|------|-------|----------|
| `player_private` | Player | Founded companies |
| `player_public` | Public (IPO'd) | Player's public companies |
| `ai_private` | AI CEO | AI-founded companies |
| `ai_public` | Stock market | Independents |
| `kingdom_state` | Kingdom | Royal monopolies |
| `kingdom_public` | Public + kingdom stake | Partially privatized |
| `guild` | Trade guild | Craft/market guilds |

### AI CEO System
- AI CEOs found companies independently
- AI companies compete with player
- AI can IPO, expand, attempt takeovers
- AI personality affects strategy (aggressive/conservative/innovative)

```javascript
aiCEO = {
  id: 'ceo_01',
  name: 'Lord Ashford',
  personality: 'aggressive',  // aggressive/conservative/innovative
  wealth: 150000,
  companies: [0, 3, 7],       // company IDs
  territory: 'Aurelia',
  relations: {}               // with other CEOs + player
}
```

### Kingdom Companies
- Kingdoms own state enterprises
- Royal monopolies on specific goods (e.g., Aurelia = grain monopoly)
- Can be privatized (player buys shares)
- Generate revenue for kingdom treasury

## 4. TAXATION SYSTEM

### Tax Layers
```
1. Kingdom Tax      → on all economic activity in territory
2. City Tax         → additional local tax
3. License Fee      → operating in territory
4. Trade Tariff     → cross-border transactions
5. Corporate Tax    → on company profits
6. Personal Tax     → on player gold (wealth tax)
```

### Tax Rates by Government
| Government | Corp Tax | Trade Tariff | License Fee |
|------------|----------|--------------|-------------|
| Monarchy | 25% | 15% | High |
| Parliament | 15% | 8% | Medium |
| Theocracy | 20% | 10% | Medium |
| Technocracy | 8% | 5% | Low |
| Barony | 30% | 20% | Very High |
| Merchant Rep | 10% | 3% | Low |
| Junta | 20% | 12% | Medium |
| City-State | 5% | 2% | Low |

## 5. FACTIONS & INTERNAL POLITICS

Each kingdom has internal factions that affect policy:

| Faction | Agenda | Effect |
|---------|--------|--------|
| **Royalists** | Strengthen crown | Higher taxes, more armies |
| **Merchants** | Free trade | Lower tariffs, open borders |
| **Clergy** | Religious law | Theocracy drift, tithes |
| **Military** | Expansion | War inclination, arms spending |
| **Populists** | People power | Lower taxes, higher stability |
| **Technocrats** | Innovation | Research grants, lower corp tax |
| **Nobles** | Land rights | Feudal drift, high license fees |

### Faction Power
- Factions gain/lose power based on events
- Dominant faction influences government decisions
- Revolutions possible if stability drops

## 6. DIPLOMACY EXPANSION

### New Actions
| Action | Cost | Effect |
|--------|------|--------|
| Trade Treaty | Relations 40+ | 0% tariffs between nations |
| Defense Pact | Relations 60+ | Mutual defense |
| Vassalage | Military power | Subordinate kingdom |
| Trade Embargo | — | Block all trade |
| Annexation | War victory | Absorb territory |
| License Agreement | $10K | Automatic license for all cities |

### Political Marriages (Monarchies)
- Alliance through marriage between royal families
- +30 relations, inheritance claims possible

## 8. DATA MIGRATION

### Save Format v5 → v6
- gov: string → gov: {type, taxRate, stability, ...}
- companies: add ownerType field
- countries: add territoryTiles array
- player: add licenses array
- new: aiCEOs array

### Backward Compatibility
- Old saves: gov string → upgrade to {type: oldValue, ...defaults}
- Companies without ownerType → default to 'public'
- No territory → generate from city positions
