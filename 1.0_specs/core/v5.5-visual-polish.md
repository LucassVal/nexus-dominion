### Data
```
BUTTON_STATES = {
  city:     { need: 'nearCity',  tooltip: 'Precisa estar perto de uma cidade' },
  trade:    { need: 'nearCity',  tooltip: 'Mercado disponivel nas cidades' },
  travel:   { need: 'nearCity',  tooltip: 'Trens e navios partem das cidades' },
  camp:     { need: 'notMoving', tooltip: 'Pare de andar para acampar' },
  diplomacy:{ need: 'gameActive',tooltip: 'Disponivel apos iniciar o jogo' },
  finance:  { need: 'gameActive',tooltip: 'Disponivel apos iniciar o jogo' },
}
```

### Mechanics
1. Cada botao HUD/topBar verifica seu `need` antes de renderizar
2. Se nao atende: botao fica cinza (opacity 0.4) + atributo title
3. Se atende: cor normal + sem tooltip de bloqueio
4. Funcao `canAct(action)` retorna bool + reason
5. Aplicar nos botoes: City, Trade, Travel, Camp
