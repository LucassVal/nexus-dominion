## Interaction Functions

| Function | HUD/Trigger | Description |
|----------|-------------|-------------|
| `interactCity()` | C key / HUD button | Find nearest city, open modal |
| `healAtCity()` | City modal | Pay $50, restore full HP |
| `joinArmy(country)` | City modal | Enlist, get $200, trigger combat |
| `campAction()` | HUD button | Toggle camping, heal 2 HP/tick |
| `zoomMap(delta)` | HUD buttons / scroll | Adjust zoom 0.12-2.0 |
| `saveGame()` | Top bar / auto | Serialize to localStorage |
| `loadGame()` | Start screen | Deserialize from localStorage |
