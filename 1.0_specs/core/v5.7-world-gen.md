# SPEC: World Generation & Terrain v5.7

## Status: ✅ IMPLEMENTED

## Map
- 60×45 tiles at 40px each (2400×1800 world)
- 5 biomes: Water, Plains, Forest, Mountain, Desert
- Value noise generation with cosine interpolation
- Biome blending (smooth borders between adjacent biomes)
- Beach strips (sand where water meets land)

## Terrain Types
| ID | Name | Color | Speed Mod | Decorations |
|----|------|-------|-----------|-------------|
| 0 | Water | #1a3a5c | 0 (blocked) | Waves (animated) |
| 1 | Plains | #3a6b35 | 1.0 | — |
| 2 | Forest | #1e4a1e | 0.7 | Trees (35% density) |
| 3 | Mountain | #4a4a4a | 0.4 | Rocks (20% density) |
| 4 | Desert | #c4a64a | 0.8 | Rocks (20% density) |

## World Entities
- 24 cities (4 per country, 6 countries)
- Roads connecting cities within same country
- Train routes: cities >150K population
- Ship routes: coastal cities across countries
- 10 bandits (random placement)
- 6 AI traders (start in random cities)
- 1-3 armies per country
- City types: village (<50K), town (<200K), city (<1M), metro (>1M)

## City Properties
- Population, GDP, stability, owner country
- Building silhouettes (procedural)
- Facilities (auto-generated at world gen)
- Supply/demand/prices for all goods
- City type affects: ring size, building count, facility slots

## Terrain Features
- Terrain cache (offscreen canvas for performance)
- Water animation (sinusoidal offset)
- Tree positions (forest tiles, random)
- Rock positions (mountain/desert tiles, random)
- Day/night cycle: hour = (day×4 + tick)%24

## Functions
| Function | Description |
|----------|-------------|
| `genTerrain(seed)` | Generate terrain + cache + decorations |
| `genWorld()` | Generate cities, countries, roads, routes |
| `getTile(x,y)` | Get terrain type at world position |
| `buildTCache()` | Build terrain cache canvas |

## Implementation
- Functions: genTerrain, genWorld, getTile, buildTCache
- Data: terrain[45][60], terEle[45][60], tCache, trees[], rocks[], stars[]
- Constants: COLS=60, ROWS=45, TILE=40, T[5], TC[5], TS[5]
