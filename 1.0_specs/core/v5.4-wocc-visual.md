## FEATURE 1: PROCEDURAL ENTITY ICONS (Canvas-drawn, no emoji)

### Substituir emoji/texto por icones desenhados via Canvas:

| Entidade | Atual | Novo (Canvas-drawn) |
|---|---|---|
| Player | 🔺 triangulo dourado (ok) | Mantem + outline mais grosso + glow animado |
| Bandits | ☠ texto 6px | 5px circulo vermelho + "X" preto dentro |
| AI traders | bolinha verde 4px | 4px circulo verde + cifrao "$" branco dentro |
| Army | bolinha vermelha 7px + letra | escudo 7px com cor do pais |
| Cities | circulo colorido | mantem + anel do tipo (village/town/city/metro) |

### Mechanics
1. Desenhar cada entidade com `ctx.arc()` + `ctx.fillRect()` + `ctx.stroke()`
2. Bandit: circulo vermelho escuro + "X" branco (2 linhas cruzadas)
3. Trader: circulo verde + "$" via fillText pequeno
4. Army: forma de escudo (arc + rect combinado) com cor do pais

## FEATURE 2: BIOME BLENDING (Terrain transitions)

### Atual: tiles com cores solidas, borda preta 0.5px
### Novo: blur/smooth entre biomas adjacentes

### Mechanics
1. Ao renderizar terrain cache, interpolar cores nas bordas entre biomas
2. Transicao suave: PLAINS→FOREST (verde claro→verde escuro)
3. WATER→PLAINS: faixa de areia (beach) de 1-2 tiles
4. MOUNTAIN→PLAINS: faixa de foothills (cinza claro)

## FEATURE 3: TOWN DISTRICTS (Timber-framed WoCC style)

### Atual: predios retangulares coloridos aleatoriamente ao redor da cidade
### Novo: 3 distritos visiveis por tipo de cidade

| Tipo | Distritos | Visual |
|---|---|---|
| Village | 1 centro | 3-5 casas madeira (marrom) ao redor de poco |
| Town | 2 distritos | mercado + casas, muralha baixa |
| City | 3 distritos | castelo/palacio + mercado + casas, muralha alta |
| Metropolis | 4 distritos | arranha-ceus + fabricas + palacio + muralha dupla |

### Mechanics
1. Gerar distritos deterministicos por cidade
2. Cada distrito: cluster de 3-8 predios com tema
3. Muralha: linha cinza ao redor (cidade/metropole)
4. Poco/chamine: indicador visual no centro

## FEATURE 4: ENTITY MOVEMENT SMOOTHING (Lerp)

### Atual: entidades teleportam em passos discretos
### Novo: interpolacao suave (lerp) entre posicoes

### Mechanics
1. Em cada frame de render: `entity.x += (entity.targetX - entity.x) * 0.1`
2. Player ja usa isso (movePlayer com velocidade), aplicar a bandits/traders/armies
3. Remove o "steps" visivel, movimento fica fluido
