## Referencias Google 2026

### Ref 1: Material Design 3 (m3.material.io)
- **Fonte**: Google's latest open-source design system
- **Aplicado**: CSS variables com naming consistente, tonal surfaces, rounded corners (12-16px)
- **ND**: `:root` com `--accent`, `--gold`, `--muted`, radius 7-16px nos elementos

### Ref 2: Codexical — Glassmorphism Accessibility (Abril 2026)
- **Alerta**: WCAG 2.2 requer 4.5:1 contraste — glassmorphism falha por definicao
- **Solucao ND**: Texto com `text-shadow` duplo (stroke effect) para legibilidade
- **ND**: Titulos com `text-shadow: 0 0 10px cyan`, botoes com `text-shadow: 0 1px 2px rgba(0,0,0,0.5)`
- **Fallback**: ND usa glass apenas em overlays (HUD, modais) — o mapa (background) e Canvas puro

### Ref 3: Neel Networks — Glassmorphism Guide 2026
- **CSS Canonico**:
```css
.glass-card {
  background: rgba(255,255,255,0.15);
  backdrop-filter: blur(12px);
  border: 1px solid rgba(255,255,255,0.25);
  border-radius: 16px;
  box-shadow: 0 8px 32px rgba(0,0,0,0.12),
              inset 0 1px 0 rgba(255,255,255,0.2);
}
```
- **ND**: Implementado com dark mode adaptation (`rgba(13,19,32,0.65)` em vez de white)

### Ref 4: UX Pilot — 12 Glassmorphism Features
- **Best practices**: layered depth, light reflection, Z-depth, subtle borders
- **ND**: `box-shadow` com `inset 0 1px 0 var(--glass-highlight)` simula light reflection
- **ND**: Multiplas camadas (topbar → HUD → modal → overlay) com z-index escalonado
