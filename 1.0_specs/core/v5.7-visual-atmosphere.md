# SPEC: Visual Polish v5.7 — Background & Atmosphere

## Status: ✅ IMPLEMENTED

## Background System

### Starfield
- 200 static stars generated at world creation
- Properties: x, y, size (0.3-1.8px), brightness (0.3-1.0)
- Rendered with 20% camera parallax (subtle depth effect)
- Flicker animation: `0.5 + sin(tick*0.05 + star.x) * 0.5`
- Color: rgba(200, 220, 255, brightness × flicker)

### Map Border Glow
- Radial gradient from center to edges
- Inner: transparent (map visible)
- Outer: rgba(0,0,0,0.7) dark fade
- Creates "spotlight" effect on the game world

### Vignette
- Always-on subtle dark edges
- Radial gradient covering screen corners
- Opacity: 0.55 at edges, 0 at center
- Color: rgba(0,0,8,0.55)

### Day/Night Cycle (existing, maintained)
- Hour = (day × 4 + tick) % 24
- Night: hours 20-6
- Blue overlay: rgba(5,5,30, night × 0.45)
