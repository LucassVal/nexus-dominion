# SPEC: Bandit & Corsair Mode v6.5

## Status: 📝 SPEC → IMPLEMENTING

## BANDIT ROLE (fora-da-lei)

### Start
- Role card: 🦹 Bandit — $50, bounty $500, 0 relations
- Start in forest/mountain tile (safe zone)
- Can't enter cities normally (guards attack)

### Gang System
```javascript
player.gang = [
  {id:'g0', name:'Scar', hp:40, atk:8, def:3, loyalty:70},
  ...
]
```
- Recruit: defeat bandits → chance to recruit (30% + notoriety/5)
- Max gang: 3 + floor(notoriety/15)
- Gang fights alongside in combat
- Loyalty decays if no loot share, low loyalty = betrayal

### Notoriety
- 0-100 scale
- +5 per successful heist
- +2 per trader robbed
- +1 per combat won as bandit
- Bounty hunters spawn more frequently at high notoriety
- Unlocks: more gang slots, better heists, hideout upgrades

### Hideouts
- Found in forest/mountain tiles (click to establish)
- Safe zone: heal, store items, save game
- Max 3 hideouts
- Can be discovered by patrols (low chance)

### Heists
- Available at hideouts
- Types: trader robbery, caravan raid, bank job, train heist
- Risk/reward: higher notoriety = better heists
- Fail = jail time (skip N days, lose gold)

### City Entry
- Bribe guards: $100 + bounty/10
- Disguise: stealth check (skill × 10% chance)
- Once inside, can trade normally

## CORSAIR (naval variant)
- Bandit + ship focus
- Start near coast
- Can raid ships, coastal towns
- Own a ship (upgradeable)
- Naval combat encounters
