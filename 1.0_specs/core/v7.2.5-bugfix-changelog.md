# v7.2.5 Bugfix Changelog — Neocortex V44 Chain-Thinking RCA

> 9 bugs corrigidos via metodologia Neocortex V44 (R21 + 5 Porques + 13 passos por nivel).

---

## 🔴 CRITICAL (funcionalidade quebrada)

### BUG-001: `cfgDiff` difficulty selector phantom
- **Sintoma**: Dificuldade sempre 'medium'
- **RCA**: DDD refactor adicionou leitura do DOM sem elemento HTML
- **Fix**: `<select id="cfgDiff">` no World Config grid

### BUG-002: `loadGame()` loses mapType  
- **Sintoma**: Saves com archipelago/pangea/highlands viram 'default'
- **RCA**: G.mapType nunca armazenado/serializado/restaurado
- **Fix**: Store + serialize + restore + passar pra genTerrain

## 🟡 MEDIUM

### BUG-003: Redirect cache-bust destroys query params
- **Sintoma**: `?eruda=true` apagado
- **RCA**: `location.search='?v='` substitui, nao append
- **Fix**: `location.search+=(search?'&':'?')+'v='`

### BUG-004: tbSpd/tbMount dynamic-only
- **Sintoma**: Elementos criados via JS, ausentes do HTML
- **Fix**: Adicionados spans no topBar estatico

## 🟢 LEVE

### BUG-005: 16 var → 12 let
- **Fix**: updateUI, start, shakeScreen, updateNewsTicker

### BUG-006: Console interceptor offline-first
- **Sintoma**: 8640 fetch/dia para servidor offline
- **Fix**: NDLog.enableServer() explicito

## 🔴 RUNTIME CRASHES

### BUG-007: `spdColor is not defined` in mdTransport()
- **Sintoma**: Crash ao abrir Transport modal
- **RCA**: spdColor/effSpd referenciados mas nao calculados
- **Fix**: getTile + terrain lookup + color tier

### BUG-008: `ct is not defined` in mdCity()
- **Sintoma**: Crash ao abrir City modal
- **RCA**: ct.type referenciado mas ct nunca definido (nc e a cidade)
- **Fix**: ct.type → getCityType(nc.pop)

### BUG-009: NPCs outside map bounds (dynamic maps)
- **Sintoma**: Bandits/traders/armies fora das bordas em mapas nao-default
- **RCA**: COLS(60)/ROWS(45) hardcoded vs genTerrain dinamico (MC/MR)
- **Fix**: 5 pontos: GLOBE.wrapX, warpEntity, bandit spawn, hideout spawn, trader Y clamp
