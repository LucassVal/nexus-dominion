## CORE LOOP (Mount & Blade 2)

```
START: Nobody (wanderer)
  ↓
RECRUIT: troops from villages
  ↓
FIGHT: bandits, looters, enemy lords → XP + loot
  ↓
LEVEL: skills, equipment, renown
  ↓
MERCENARY: serve a kingdom, earn gold
  ↓
VASSAL: swear loyalty → get a FIEF (village/castle)
  ↓
MANAGE: upgrade fief (buildings, tax, garrison)
  ↓
KING: start own kingdom or rebel
  ↓
CONQUER: take enemy fiefs → expand realm
```

## EXPANSAO 1: Victoria 3 (Economia)

### POP System
```javascript
city.pops = {
  farmers: 5000,      // produce food
  laborers: 2000,     // produce goods
  artisans: 800,      // processed goods
  merchants: 400,     // trade value
  aristocrats: 100,   // tax income
  clergy: 200         // stability
}
```

### Production Chains
```
Farm (grain) → Mill (flour) → Bakery (bread)
Mine (iron_ore) → Smelter (iron) → Workshop (tools)
Pasture (wool) → Weaver (cloth) → Tailor (clothes)
```

### Market Prices (supply/demand)
- Cada cidade tem supply/demand (ja existe!)
- Precisa conectar com POPs: mais POPs = mais demanda
- Shortage = preco sobe, surplus = preco cai

## EXPANSAO 2: EU5 (Provincias + Diplomacia)

### Province System
```
KINGDOM
  └── PROVINCE (regiao com 3-8 locations)
       └── CITY (capital da provincia)
            └── VILLAGE (produz recursos para a city)
```

### EU5 Diplomacy
- Casus belli (motivo de guerra)
- War exhaustion
- Peace deals (take provinces, gold, vassalize)
- Coalitions (reinos se unem contra agressor)
- Aggressive expansion (penalidade por conquistar muito)

## EXPANSAO 3: Civilization (Tech + Eras)

### Tech Tree
```
ERA 1 (Ancient):    Agriculture → Mining → Bronze Working
ERA 2 (Classical):  Iron Working → Writing → Currency
ERA 3 (Medieval):   Feudalism → Guilds → Navigation
ERA 4 (Renaissance): Gunpowder → Printing → Banking
ERA 5 (Industrial): Steam Power → Railways → Factories
ERA 6 (Modern):     Electricity → Flight → Computing
```

### Era Effects
- Cada era desbloqueia novas buildings, units, ships
- Era avanca com tech points (ganhos por pesquisa + trade)
- AI kingdoms tambem avancam

## EXPANSAO 4: Anno 1800 (Naval + Trade Routes)

### Ship Types (comercial)
| Ship | Era | Cargo | Speed | Use |
|------|-----|-------|-------|-----|
| Schooner | 1 | 50t | 4x | Scout |
| Clipper | 2 | 100t | 5x | Fast trade |
| Cargo Ship | 3 | 300t | 2x | Bulk |
| Tanker | 4 | 200t | 2.5x | Oil/Liquid |
| Great Eastern | 5 | 500t | 1.5x | Mega cargo |

### Trade Routes
- Definir rota: Porto A → Porto B → Porto C
- Ship automaticamente faz a rota
- Carga definida por slot (pickup→dropoff)
- Pirates podem atacar (evento aleatorio)

### Island Resources
- Ilhas no oceano com recursos unicos
- So acessiveis por navio
- Colonizar ilha = novo settlement
