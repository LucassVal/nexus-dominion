## 2. KINGDOM CREATION MODE (Alternative Start)

Instead of picking a role, player can **found a kingdom**:
- Choose capital location on map
- Name the kingdom
- Pick government type
- Design banner color
- Auto-generate 5 cities around capital
- Start as King role with that kingdom

### Flow:
```
Start Screen → [Found Kingdom] button
  → Name input
  → Government picker (8 types)
  → Color picker (6 preset colors)
  → Map preview (click to place capital)
  → Auto-gen 5 cities in radius
  → Start game as King of new realm
```

## 4. PLAYER-FOUNDED KINGDOM

### Data
```javascript
playerKingdom = {
  name: 'Myrealm',
  color: 'rgba(x,x,x,1)',
  capital: cityId,
  founded: day,
  cities: [cityIds],
  gov: {type, name, ic, taxRate, ...},
  treasury: 200000,
  mil: 30,
  army: [{id, sz, x, y}],
  recognized: [] // countries that recognize this kingdom
}
```

### Diplomatic Recognition
- Other kingdoms may or may not recognize player kingdom
- Recognition depends on: relations, military power, time
- Unrecognized = can't trade, can be attacked freely
- Send envoys to gain recognition ($5K per country)
