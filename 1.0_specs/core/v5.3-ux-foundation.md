# SPEC: Nexus Dominion v5.3 — UX Foundation

## BUGS CRITICOS (bloqueiam jogo)

### B1: HUD botoes nao funcionam
- Botoes: City, Trade, Items, Skills, Travel, Camp
- Causa provavel: funcoes chamadas por onclick nao definidas ou G nao inicializado
- Fix: verificar cada onclick → function mapping, adicionar guards

### B2: Player invisivel no minimapa  
- Player dot (amarelo) pode estar fora da viewport do minimapa
- Ou cor igual ao terreno
- Fix: dot vermelho pulsante, sempre visivel, borda branca

### B3: Fonte ilegivel no mapa
- Nomes de cidades sem contraste contra o terreno
- Texto preto/cinza some em biomas escuros
- Fix: stroke branco ao redor do texto (textShadow no Canvas)

### B4: Sem menu Settings/Restart
- Jogador nao consegue reiniciar sem fechar browser
- Nao tem como ajustar volume, zoom, etc
- Fix: modal Settings com: New Game, Volume, Zoom, DevTools toggle
